О проекте

Chatlab — это учебный проект, демонстрирующий принципы построения чат-бота на основе ChatGPT, а также модуль генерации изображений. Проект предназначен для демонстрации работы с системами контроля версий, документированием кода и созданием справочной документации.

Все возможности распределены по модулям, что упрощает разработку и тестирование.

Модули проекта

1. Модуль интерфейсов

Обеспечивает консольный (и потенциально веб) интерфейс для взаимодействия с Chatlab.

2. Модуль интеграции с ChatGPT

Связывается с (псевдо) ChatGPT, передавая запрос и возвращая ответ.

3. Модуль генерации изображений

Имитирует создание изображения на основе текстового описания.

4. Модуль управления контекстом

Хранит историю сообщений, чтобы бот учитывал предыдущие реплики пользователя.

5. Модуль авторизации и ролей

Разделяет пользователей (гость, пользователь, админ) и даёт разные права.

6. Модуль мультиязычной поддержки

Определяет язык вопроса и возвращает ответ на том же языке (заглушка).

7. Модуль аналитики и логирования

Записывает логи, подсчитывает статистику (часто задаваемые вопросы, количество запросов).

8. Модуль интеграции с мессенджерами

Потенциально позволяет связать Chatlab с Telegram, Slack и др.

9. Модуль тестирования и отладки

Содержит тесты для проверки корректности работы всех частей.

10. Модуль резервного копирования

Делает бэкапы истории, логов и настроек, чтобы восстановить систему при сбое.

Как использовать Chatlab

  1. Скачать репозиторий с GitHub или любым другим способом.
  2. Убедиться, что Python 3.x установлен (для консольной версии).
  3. Открыть терминал и перейти в папку проекта.
  4. Запустить client_interface.py, чтобы начать диалог с ботом.
  5. При необходимости использовать команду image <описание> для генерации (имитации) изображения.
